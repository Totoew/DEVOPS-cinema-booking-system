### Предварительные требования (что установить)

1. Node.js 20

### Шаги сборки и запуска бэкенда

1. Откройте терминал
2. Выполните npm install
2. Запустите npm run dev

API доступно на http://localhost:4000

### Шаги сборки и запуска фронтенда

1. Откройте терминал
2. Выполните npm install
3. Запустите npm start

Приложение доступно на http://localhost:4200

См. файл SOLUTION.md для полного образа решения и требований к ДЗ.
